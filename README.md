# 🛵 Sistema de Delivery

Sistema de delivery multi-estabelecimento com suporte a Açaí, Pizzaria, Hamburgueria e muito mais.

## Stack
- **Frontend**: React + Vite + Styled Components
- **Backend**: Node.js + Express (MVC)
- **ORM**: Sequelize (SQLite para dev / PostgreSQL para prod)
- **Auth**: JWT

## Estrutura
```
sistema_delivery/
├── backend/       # API REST (Node.js + Express + Sequelize)
└── frontend/      # React App (Vite)
```

## Quick Start

```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (outro terminal)
cd frontend && npm install && npm run dev
```
